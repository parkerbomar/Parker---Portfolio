FITLOCK

Prototype - PDF - Presentation

Intro

June 2018 - While taking an Interactive Design class at Kennesaw State University, my team and I created this project during a one-week sprint. The goal was to create an app, presentation, and PDF and then iterate on it after conducting user testing, to make the app more focused on goal-oriented design. As the team lead and lead designer, I put in hard work and a clear vision to get the project done by the deadline, but it turned about amazing.

*User group was based on class demograpic.

 

 

 


THE PROBLEM

Millions of people aged 19-33* around the world each day want to be fit and healthy but lack the motivation to do so.  They have the desire to work, but put off the actual work out.
 
THE SOLUTION

Create an app which motivates the user, so the user does not have to rely on internal motivation.

WE ARE FITLOCK

FitLock is an app which allows anyone who lacks motivation to have the drive to get fit.  It allows the user to set an alarm that can only be deactivated after the exercise is complete.
